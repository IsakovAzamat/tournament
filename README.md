# 🎮 MLBB Scrim Platform (Backend)

Backend сервис для организации матчей (праков) между командами Mobile Legends: Bang Bang.

Проект разрабатывается как учебно-практическая backend система с авторизацией, командами и матч-логикой.

---

## ⚙️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT
- PostgreSQL
- Lombok
